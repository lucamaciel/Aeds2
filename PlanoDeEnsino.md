## 📋 Descrição da Disciplina

Este repositório documenta o progresso acadêmico em Algoritmos e Estruturas de Dados II, com foco no desenvolvimento de programas eficientes através de estruturação de dados e aplicação de algoritmos de ordenação e pesquisa. A disciplina enfatiza construção, teste, documentação e análise crítica de soluções computacionais, com aprendizagem centrada em resolução de problemas e manipulação de dados em memória dinâmica.

```
FQ = 1 + 0.3 × (Soma Quizzes / 6)

Exemplo: 4.5 pts de quizzes → FQ = 1 + 0.3 × 0.75 = 1.225
```
